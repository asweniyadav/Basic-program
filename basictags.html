The Basic HTML tags.

This is the first basic html page.
I made this page by using the basic HTML tags.
The tags used to develop/generate this page are as follows :-

  • html
  • head
  • title
  • body
  • h1 (header tag used to add heading to a particular topic)
  • p (paragraph)
  • br (used to break a line)
  • strong (used to show any text area in bold)
  • ul and li (used to list any item)
